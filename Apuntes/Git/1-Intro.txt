COMMIT:
-------
Consolidar, confirmar​ o hacer un commit se refiere, en el contexto de la ciencia de la computación y la gestión de datos, a la idea de confirmar un conjunto de cambios provisionales de forma permanente. 

FETCH:
-----
Fetch es el nombre de una nueva API para Javascript con la cuál podemos realizar peticiones HTTP.

PULL:
-----
Hace referencia a un sistema que se ajusta en todo momento a la demanda. Esto quiere decir que no se produce nada hasta que no hay una demanda real del producto.

PUSH:
-----
Una programación de tipo push es una programación no referenciada que puede generarse mediante un sistema de planificación o crearse manualmente.
